Objectif de la Mission

Avant l'application

  • Récupérer la vidéo
  • Séparer la vidéo avec FFmpeg
  • Analyser chaque image avec une macro FiJi
  • Stocker les résultats dans un fichier Excel
  • Combiner ces résultats avec les fichiers que la machine sortait
  • Répéter l'opération pour chaque vidéo et faire une moyenne

Après l'application

  • Lancer l'application
  • Sélectionner les différentes videos
  • Saisir les paramètres utiles
  • Lancer l'analyse
